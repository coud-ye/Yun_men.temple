# 云门寺寺院事务管理系统（公开演示版）

访问地址：https://coud-ye.github.io/Yun_men.temple/

这是一个部署在 GitHub Pages 上的静态前端演示项目，用于展示寺院人员资料、请假、线上登记和管理员审批工作台的交互流程。

## 演示入口

- 管理员：`demo-admin` / `demo123456`
- 人员：`demo-personnel` / `demo123456`
- 访客：在登录页点击“我是访客，进入线上登记”。

## 角色说明

- 管理员可使用工作台、审批中心、人员资料和模拟账号创建功能。
- 人员仅可查看或修改个人资料、提交请假并查看记录。
- 访客仅可使用线上登记入口，无法访问管理资料和审批内容。

## 演示版限制

- 所有数据只在当前浏览器会话中模拟，不会提交到服务器或数据库。
- 请勿填写真实身份证号、电话、证件图片或其他敏感个人信息。
- Excel、PDF、图片上传等功能均为浏览器本地演示；刷新页面后模拟状态会重置。
- 本项目不包含真实登录认证、文件存储、Word 模板生成、审批通知或数据库。

## 本地预览

直接打开 `index.html`，或用任意静态文件服务器在项目根目录预览。

## 发布

GitHub Pages 当前从 `main` 分支根目录发布。页面的静态资源使用相对路径，可在仓库子路径下正常加载。
